# 深度学习理论资源

## 🎯 目标
掌握深度学习理论基础，而非基础实现

---

## 📚 推荐书籍

### 必读经典
1. **"Deep Learning"** - Goodfellow, Bengio, Courville (2016)
   - 深度学习圣经
   - 免费: https://www.deeplearningbook.org/
   - 重点：第5-8章（优化）、第9-11章（CNN/RNN）、第14-15章（表示学习）

2. **"Neural Networks and Learning Machines"** - Simon Haykin
   - 神经网络理论
   - 链接: https://www.amazon.com/Neural-Networks-Learning-Machines-Comprehensive/dp/0131471392

3. **"Mathematics for Machine Learning"** - Deisenroth et al.
   - 免费: https://mml-book.github.io/

### 参考
4. **"Information Theory and Statistical Learning"**
   - 信息论视角

## 📄 关键论文

### 表达能力
1. **"Approximation Capabilities of Multilayer Feedforward Networks"** - Hornik, 1991
   - 万能逼近定理

2. **"Representation Benefits of Deep Feedforward Networks"** - Telgarsky, 2015
   - 深度优势

### 优化
3. **"On the Difficulty of Training Recurrent Neural Networks"** - Pascanu et al., 2013
   - 梯度问题

4. **"Identity Mappings in Deep Residual Networks"** - He et al., 2016
   - ResNet理论

### 泛化
5. **"Understanding Deep Learning Requires Rethinking Generalization"** - Zhang et al., 2017
   - https://arxiv.org/abs/1611.03530
