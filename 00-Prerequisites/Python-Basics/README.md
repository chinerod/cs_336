## 📚 推荐书籍

### 必读
1. **"Fluent Python"** - Luciano Ramalho (2nd Edition, 2022)
   - Python进阶
   - 数据模型、面向对象、元编程
   - 链接: https://www.amazon.com/Fluent-Python-2nd-Concise-Programmers/dp/1492056359

2. **"High Performance Python"** - Gorelick & Ozsvald
   - 性能优化
   - Cython, Numba, 并行
   - 链接: https://www.amazon.com/High-Performance-Python-Performant-Programming/dp/1492055026

3. **"Python for Data Analysis"** - Wes McKinney
   - Pandas, NumPy
   - 数据处理实战
   - 链接: https://wesmckinney.com/book/

### 参考
4. **"Clean Code"** - Robert Martin
   - 代码质量
   - 链接: https://www.amazon.com/Clean-Code-Handbook-Software-Craftsmanship/dp/0132350882

5. **"Design Patterns"** - Gamma et al.
   - 设计模式
   - 链接: https://www.amazon.com/Design-Patterns-Elements-Reusable-Object-Oriented/dp/0201633612
