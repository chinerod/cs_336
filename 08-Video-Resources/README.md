## 📖 学习建议

### 观看方法

1. **预习**
   - 先看讲义
   - 标记不懂的地方
   - 带着问题看

2. **观看**
   - 1.5-2倍速
   - 暂停思考
   - 做笔记

3. **复习**
   - 整理笔记
   - 实现代码
   - 做练习

### 笔记模板

```markdown
## 视频: [标题]
**日期**:
**时长**:

### 核心观点
1.
2.
3.

### 关键公式/代码
```

-
